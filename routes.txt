//USERS//

http://localhost:3333/api/users/                    -> listar usuarios
http://localhost:3333/api/users/create              -> criar usuarios
http://localhost:3333/api/users/delete/:id          -> deletar usuarios pelo id
http://localhost:3333/api/users/login               -> fazer login 
http://localhost:3333/api/users/logout              -> fazer logout
http://localhost:3333/api/users/edit                -> editar usuario
